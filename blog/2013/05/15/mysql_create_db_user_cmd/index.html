Mysql建库及用户命令实例

May 15th, 2013 | Comments

这里记录下mysql命令行下创建库及用户的命令

>mysql -u root -p

>show databases;

>create database new_database;

>create user 'name' identified by 'pwd';

>grant all privileges on new_database.* to 'name'@'localhost' identified by 'pwd' with grant option; 

>quit

>mysql -u name -p

>show databases;
